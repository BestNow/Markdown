# 动作模式

## 胸

| 胸   | 动作                                                    |
| ---- | ------------------------------------------------------- |
| 推胸 | 杠铃卧推 哑铃卧推 双杠臂屈伸 俯卧撑 史密斯卧推 器械推胸 |
| 夹胸 | 蝴蝶机夹胸 龙门架夹胸                                   |

**推胸VS夹胸**
	推胸压迫和刺激能力更强
	推胸是复合动作可以发挥肌肉协同
	三分化组数有限基本不能让给夹胸
	夹胸可做两三组作为胸部热身的一部分
